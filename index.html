Dev Inu Token

($DINU)

DINU is an deflationary* ERC20 token on the devETH network. The token was coded and deployed for learning and for fun. Not for speculation. 99% of the supply is locked in devethswap liquidity (set 6% slippeage for swap to allow token burning.)
*4% burn, 1% tax every transaction.
**Contract code is public.
